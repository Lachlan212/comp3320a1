All test cases are found within A1/demo/src/test/java/
They are split by Question Number, and for Q2, Q3, they are divided further by function.

Project is a maven project, so testing can be run by issuing the following command into cmd/powershell: mvn clean test
NOTE: cmd/powershell window must be in the A1/demo folder
NOTE: 4 test are designed to fail. They are testing invalid equivalence partitions.
